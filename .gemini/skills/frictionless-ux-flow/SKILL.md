# Frictionless UX Flow Guidelines

1. **Clear Action Hierarchy**:
   - Primary action (e.g. Save, Submit): Solid gradient button with glow.
   - Secondary action (e.g. Cancel, Back): Muted outline or ghost button.
   - Destructive action (e.g. Delete): Subtle red tone (`#ef4444`).

2. **Form Interaction UX**:
   - Provide real-time validation visual state (green accent for valid, subtle red glow for error).
   - Display contextual error tooltips below inputs instead of disruptive popups.
   - Automatically focus the first input on load.

3. **Status Badges & Feedback**:
   - Use pill badges for status (`Active`, `Pending`, `Draft`) with subtle background tints and dot indicators.
